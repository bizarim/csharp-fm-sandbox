## fmDLL로 server application 만들기

```sh
│  README.md
├─fm-dlls
│  ├─fmCommon (클라,서버 공통 dll)

│  ├─fmServerCommon (서버 공통 dll)
│  └─fmThirdparty

   ├─appAuthServer (인증서버)
   ├─appCenterServer (센터서버)
   ├─appChatServer (채팅서버)
   └─aappGameServer (게임서버)

```
